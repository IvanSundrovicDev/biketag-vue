The New BikeTag App now enables BikeTag Players to create a Profile using social logins and to reserve your unique BikeTag name! More features will become available for the BikeTag Player Profile as development allows. Please consider contributing to the BikeTag Project or becoming a subscribing member and get special perks! BikeTag Ambassadors can also log into this application and use the administrative features for running BikeTag Games. If you have any questions or concerns, please send us an email at ambassadors@biketag.org.
